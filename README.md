# citas-react
 Proyecto administracion de pacientes en Vite, Tailwind CSS y LocalStorage del curso Udemy

Creacion proyecto:

npm init vite@latest 

Project name: citas-react
Framework: react
variant: react

cd citas-react
npm install
npm run dev
